# 敏感字段清单

## 1. 用户隐私字段

常见示例：
- 手机号
- 姓名
- 会员 ID
- 用户 ID
- 设备 ID
- 身份证号
- 邮箱
- 地址
- 订单号

默认处理：
- 轻度：掩码或稳定匿名 ID
- 重度：完全泛化为匿名角色/分组

## 2. 经营敏感字段

常见示例：
- 品牌名
- 门店名
- 门店编码
- 商品内部编码
- 内部渠道名
- 策略名
- 私域标签体系
- 城市名 / 区域名
- 业务分组代号

默认处理：
- 轻度：泛化别名
- 重度：类目级抽象

## 3. 数字敏感字段

常见示例：
- 参与人数
- 触达人数
- 支付人数
- 用券人数
- GMV
- 增量 GMV
- 收入
- 成本
- ROI
- 客单价
- 单店产出
- 核销数
- 转化率
- 复购率
- 排名值

默认处理：
- 轻度：扰动
- 重度：区间 / 等级 / 量级 / 描述性表达

## 4. 文本叙事敏感内容

常见示例：
- 明确品牌名
- 明确门店或城市名
- 会暴露内部策略细节的命名
- 直接引用原始经营数字的结论
- 可联动反推真实数据的文字说明

默认处理：
- 在保留分析含义的前提下重写
